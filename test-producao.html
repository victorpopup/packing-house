Teste de Registro de Produção

Informações de Debug

Console: Abra o console do navegador para ver logs detalhados

Funções disponíveis:

  • debugProducao() - Mostra informações completas do sistema
  • testarMarcas() - Força a exibição do autocomplete
  • cadastrarMarcaTeste() - Cadastra uma marca de teste

Passos para testar:

  1. 1. Clique em "Cadastrar Marca Teste" para criar dados
  2. 2. Clique no campo "Marca" para focar
  3. 3. Digite algo para filtrar ou clique em "Testar Autocomplete"
  4. 4. Clique em uma marca para selecionar
  5. 5. Verifique se o campo foi preenchido
